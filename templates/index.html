Building a Drinks App Api with Flask

This is a Simple Api about My Favourite(40%) Drinks Powered by Flask.

It's simply a backend app hence consists of a database and flask micro-framework.

I used Sql-Alchemy as the Orm to easily connect, render, post, delete and update data.

Below are links you can use to view the data ,either by id or all in general.You can as well as manipulate the data through methods such as updating and deletion of records.

For Delete,update and Post Methods you'll have to use Postman(working with Apis tool) to manipulate the data.
